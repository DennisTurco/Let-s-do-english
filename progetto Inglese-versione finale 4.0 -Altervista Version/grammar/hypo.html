IF CLAUSES
Il periodo ipotetico si usa per descrivere la conseguenza di un'azione se si verifica una determinata condizione.
È formato da due frasi:
PRINCIPALE o MAIN CLAUSE: descrive la conseguenza + SUBORDINATA o IF CLAUSE: indica la condizione

I periodi ipotetici sono di 4 tipi:
Tipo 1 - FATTI REALI
Tipo 0 - FATTI ABITUALI
Tipo 2 - FATTI POSSIBILI
Tipo 3 - FATTI IRREALI

ZERO CONDITIONAL
Il tipo 0 descrive una conseguenza che si verifica regolarmente ogni volta che si ha una certa condizione, quindi si tratta di verità universali, condizioni abituali, processi fisici o meccanici.
IF CLAUSE- PRESENT SIMPLE+ MAIN CLAUSE- PRESENT SIMPLE
If it is too hot in June, the grass gets dry.
If I don't have the chamomile tea after dinner, I don't sleep well.

FIRST CONDITIONAL
Il tipo 1 descrive ipotesi realistiche ed eventi altamente possibili nel presente o nel futuro, le cui conseguenze sono facili da prevedere. È spesso usato per dare avvertimenti.
IF CLAUSE- PRESENT SIMPLE+ MAIN CLAUSE- SIMPLE FUTURE
If it rains, the party will take place indoors.
If you don't go shopping today, you will miss the special offer.

SECOND CONDITIONAL
Il tipo 2 esprime ipotesi poco probabili o fatti difficilmente realizzabili nel presente o nel futuro.
IF CLAUSE- SIMPLE PAST+ MAIN CLAUSE- PRESENT CONDITIONAL
If I lived on a lonely island, I would go fishing all day.
If it wasn't so damp, we would sit on the grass.

THIRD CONDITIONAL
Il tipo 3 esprime ipotesi che sarebbero o non sarebbero potute accadere nel passato a determinate condizioni. Si tratta di situazioni irreali.
IF CLAUSE- PAST PERFECT+ MAIN CLAUSE- PERFECT CONDITIONAL
If you had asked me, I would have helped you.
If it hadn't been such a delicate matter, I wouldn't have needed to talk to you.

NOTA BENE

Nella IF CLAUSE si usa WERE per tutte le persone nel linguaggio formale e WAS per la prima e la terza persona singolare nel linguaggio informale.
Nelle espressioni IF I WERE YOU, IF I WERE IN YOUR SHOES, usate per dare consigli e suggerimenti, si usa sempre were.
Il periodo ipotetico di tipo 3 è l'unico che si riferisce a fatti e condizioni del passato.
